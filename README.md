# Personal Finance Chatbot 💰🤖

A smart **Personal Finance Chatbot** built with **FastAPI (Backend)** and **React + Vite (Frontend)** that helps users track income, expenses, balances, savings, and financial insights through a conversational interface.

---

## 🚀 Features

### 💬 Chat-Based Finance Management
Use natural commands such as:

- `Add expense 500 food`
- `Add income 30000 salary`
- `Show balance`
- `Monthly summary`
- `Weekly summary`
- `Daily transactions`
- `Delete all transactions`
- `Show categories`

---

### 📊 Dashboard Analytics

- Current Balance
- Total Income
- Total Expenses
- Savings Overview
- Expense Category Breakdown (Pie Chart)
- Transaction History
- Edit/Delete Transactions

---

### 🧠 Smart Insights

- Spending summaries
- Category analysis
- Budget awareness
- Financial habit tracking

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Vite
- CSS3
- Recharts (for graphs)

### Backend
- FastAPI
- Python
- SQLite
